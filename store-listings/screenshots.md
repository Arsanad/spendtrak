## Screenshot Design Guidelines

### Visual Style
- Use actual app screenshots (not mockups)
- Maintain consistent device frame across set
- Use device frames that match the platform
- Keep status bar clean (full battery, good signal, appropriate time)

### Text Overlays
- Keep captions short (5-7 words max)
- Use SpendTrak brand fonts
- Ensure text is readable at thumbnail size
- Position text above or below device, not over important UI

### Color Palette
- Background: Dark gradient matching app theme
- Text: White or accent colors from brand palette
- Maintain consistency across all screenshots

### Do's
- Show real, realistic data (not placeholder text)
- Highlight key features with subtle callouts if needed
- Use consistent visual language across all screenshots
- Ensure screenshots tell a cohesive story in sequence

### Don'ts
- Don't show empty states or error screens
- Don't use low-quality or blurry images
- Don't include personal or sensitive information
- Don't overcrowd with too much text
