# App Store Screenshots

## iOS Required Sizes
- 6.7" (iPhone 15 Pro Max): 1290 x 2796
- 6.5" (iPhone 14 Plus): 1284 x 2778
- 5.5" (iPhone 8 Plus): 1242 x 2208

## Android Required
- Phone screenshots: 1080 x 1920 (minimum)
- Feature graphic: 1024 x 500

## Recommended Screens to Capture
1. Dashboard overview
2. Add transaction
3. Receipt scanning
4. AI consultant
5. Budget overview
6. Subscription management
7. Net worth dashboard
8. Achievements

## How to Generate
Use `npx expo run:ios` on a simulator and take screenshots,
or use Maestro for automated screenshot capture.
