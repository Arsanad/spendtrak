# Changelog

All notable changes to SpendTrak will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-02-05

### Added
- AI Financial Consultant (QUANTUM) powered by Gemini
- Receipt scanning with OCR
- Behavioral spending analysis
- Premium subscription tier via RevenueCat
- Household sharing with role-based access
- Net worth tracking (assets, debts, investments)
- Zero-based budgeting
- Gamification system (achievements, streaks, challenges)
- Email receipt import
- Multi-currency support with live exchange rates
- 12 language translations
- Offline queue with automatic sync
- Data export (CSV, PDF, JSON)
- Dark theme with neon green aesthetic

### Security
- AES-256-GCM encryption for email credentials
- Row Level Security on all user data tables
- PKCE OAuth flow
- Sentry error monitoring

## [1.0.0] - Initial Release
- Basic expense tracking
- Budget management
- Category management
